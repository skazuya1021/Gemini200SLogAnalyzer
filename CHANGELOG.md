# 変更履歴

## [1.2.1] - 2026-07-15

### 修正
- ManualLog の A 列（Time）を1行目日時からの経過秒として正しく解釈し、`yyyy/MM/dd HH:mm:ss` 形式の DateTime に変換
- ManualLog のソート日時をファイル名より1行目の作成日時を優先して使用

## [1.2.0] - 2026-07-15

### 追加
- **ManualLog（.csv）** データの読込み・合体・分析・グラフ表示に対応
- データ種別の切替（Wafer Log / ManualLog）
- ManualLog のフォルダ選択時、サブフォルダを含む `.csv` を再帰的に検索
- ManualLog のファイル名・作成日時から日付順ソート、Cassette（L04 等）の自動抽出
- ManualLog の `Time` 列を `DateTime` に変換して既存の分析・グラフ機能と統合
- ManualLog 用の前回フォルダ記憶（`LastManualLogInputFolder`）

## [1.1.0] - 2026-07-03

### 追加
- グラフ種類に **面グラフ**・**棒グラフ**・**散布図** を追加
- 散布図で X 軸・Y 軸を任意の項目（DateTime 含む）から設定可能
- 散布図で複数の X-Y 組み合わせを追加・一括追加して同時描画
- Lot ID / Recipe ID フィルターに **一括選択**・**一括解除** ボタン
- グラフ表示の **表示リセット** ボタン（全フィルター選択・折れ線グラフに復帰）
- データ表示画面で日付範囲変更時、Lot ID / Recipe ID を範囲内の値のみに自動更新
- グラフタブで Lot ID / Recipe ID / Cassette による表示フィルター
- 変更履歴（CHANGELOG.md）とバージョン表示（v1.1.0）

### 修正
- 起動時にグラフ設定の初期化でクラッシュする問題を修正
- 項目名に `[Set]` や `(Median)` を含む場合の DataGrid 表示エラーを修正

## [1.0.0] - 2026-07-01

### 追加
- Gemini200S ログファイル（.log）の読込み（複数ファイル / フォルダ選択）
- ログファイルの日付順合体と CSV 出力
- データ表示（項目選択、日付・Lot ID・Recipe ID フィルター、Median/Average/Max/Min）
- Excel 形式での分析結果保存
- 折れ線グラフ・ドットプロット表示と PNG 保存
- 並列読込みとプログレスバー表示
- 前回使用フォルダの記憶機能
