## [1.3.3] - 2026-07-22

### 修正
- UI 全体の日本語フォントを Windows 11 標準の **Yu Gothic UI** に統一（文字化け・表示崩れを修正）
- PNG 保存時の注釈テキストも Yu Gothic UI で描画

### 改善
- データ表示画面: 行仮想化により表示範囲内の行のみ描画、スクロールに応じて表示を更新
- データ表示画面: DataTable への全行コピーを廃止し、大量データでも表示開始が高速化
- グラフ画面: 2 点間差分測定で **点1を指定 / 点2を指定** を切り替え、任意の位置を個別に変更可能

## [1.3.2] - 2026-07-22

### 追加
- グラフタブにズーム連動 LOD（表示点数自動調整）を適用
- グラフタブに表示ステータス・「表示をリセット」ボタンを追加

## [1.3.1] - 2026-07-22

### 追加
- 変動確認タブに分析結果グラフを表示
- データ数に応じた LOD（表示点数自動調整）、ズームインで詳細データを表示
- 分析サンプル点のオーバーレイ表示、表示リセットボタン

## [1.3.0] - 2026-07-22

### 追加
- 2点間差分測定: 点1から指定時間（秒/分/時間）後を点2として自動マーキング
- **変動確認** タブ: 点1〜点2間の時間間隔または Y 軸値間隔での変化量分析表
- 変動分析の間隔自動設定・手動入力、CSV 出力

## [1.2.7] - 2026-07-22

### 追加
- PNG 保存時に **カーソル位置データ**・**2点間差分データ** を画像下部へ含めて保存するオプション

## [1.2.6] - 2026-07-22

### 修正
- ファイル選択・フォルダ選択・合体処理開始時に、前回の合体データ・データ表示・グラフ表示をクリア

## [1.2.5] - 2026-07-22

### 追加
- グラフ上のカーソル位置に対応する最寄りデータ点の値を表示（クロスヘア付き）
- **2点間差分測定** 機能（点1・点2をクリックして時間差・各項目の Δ を表示）
- 差分測定のクリアボタン

## [1.2.4] - 2026-07-22

### 追加
- グラフタブに **CSV として保存** 機能を追加
- 現在のグラフ表示範囲（ズーム・パン後の X 軸範囲）と、選択中の Lot/Recipe/Cassette/表示項目のみを CSV 出力

## [1.2.3] - 2026-07-22

### 修正
- ManualLog 等 DateTime 列があるデータを、ファイル単位ではなく各行の DateTime で分析・表示・グラフ描画するよう変更
- 1 秒単位・分単位など、元データの時系列解像度を保持して表示

## [1.2.2] - 2026-07-15

### 修正
- ManualLog 合体データのデータ表示・グラフ表示が空になる問題を修正
- 日付フィルターの初期値を DateTime 列の実データ範囲から算出するよう変更
- ManualLog ファイル名（`2026071400_L04.csv` 等）からの日付抽出を分析処理に統合

## [1.2.1] - 2026-07-15

### 修正
- ManualLog の A 列（Time）を 1 行目日時からの経過秒として正しく解釈し、`yyyy/MM/dd HH:mm:ss` 形式の DateTime に変換
- ManualLog のソート日時をファイル名より 1 行目の作成日時を優先して使用

## [1.2.0] - 2026-07-15

### 追加
- **ManualLog（.csv）** データの読込み・合体・分析・グラフ表示に対応
- データ種別の切替（Wafer Log / ManualLog）
- ManualLog のフォルダ選択時、サブフォルダを含む `.csv` を再帰的に検索
- ManualLog のファイル名・作成日時から日付順ソート、Cassette（L04 等）の自動抽出
- ManualLog の `Time` 列を `DateTime` に変換して既存の分析・グラフ機能と統合
- ManualLog 用の前回フォルダ記憶（`LastManualLogInputFolder`）

## [1.1.0] - 2026-07-03

### 追加
- グラフ種類に **面グラフ**・**棒グラフ**・**散布図** を追加
- 散布図で X 軸・Y 軸を任意の項目（DateTime 含む）から設定可能
- 散布図で複数の X-Y 組み合わせを追加・一括追加して同時描画
- Lot ID / Recipe ID フィルターに **一括選択**・**一括解除** ボタン
- グラフ表示の **表示リセット** ボタン（全フィルター選択・折れ線グラフに復帰）
- データ表示画面で日付範囲変更時、Lot ID / Recipe ID を範囲内の値のみに自動更新
- グラフタブで Lot ID / Recipe ID / Cassette による表示フィルター
- 変更履歴（CHANGELOG.md）とバージョン表示（v1.1.0）

### 修正
- 起動時にグラフ設定の初期化でクラッシュする問題を修正
- 項目名に `[Set]` や `(Median)` を含む場合の DataGrid 表示エラーを修正

## [1.0.0] - 2026-07-01

### 追加
- Gemini200S ログファイル（.log）の読込み（複数ファイル / フォルダ選択）
- ログファイルの日付順合体と CSV 出力
- データ表示（項目選択、日付・Lot ID・Recipe ID フィルター、Median/Average/Max/Min）
- Excel 形式での分析結果保存
- 折れ線グラフ・ドットプロット表示と PNG 保存
- 並列読込みとプログレスバー表示
- 前回使用フォルダの記憶機能
